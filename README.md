# Archomen Dotfiles

Custom Arch Linux + Hyprland setup.

## Stack

- Arch Linux
- LUKS2 + Btrfs
- systemd-boot
- Hyprland
- Athena Waybar
- Kitty
- Zsh + Starship
- Fuzzel
- SwayNC
- Hyprlock + Hypridle
- mpvpaper video wallpaper
- GTK/Qt dark theme
- Bibata cursor
- Papirus icons

## Managed configs

- Hyprland: ~/.config/hypr
- Waybar: ~/.config/waybar
- Kitty: ~/.config/kitty
- Fuzzel: ~/.config/fuzzel
- GTK: ~/.config/gtk-3.0 and ~/.config/gtk-4.0
- Qt: ~/.config/qt5ct and ~/.config/qt6ct
- Zsh: ~/.zshrc
- Starship: ~/.config/starship.toml
- Scripts: ~/.local/bin
